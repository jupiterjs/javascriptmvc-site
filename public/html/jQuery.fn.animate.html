    $.fn.animate

    jQuery.fn.animate  function     

    Source

    Animate CSS properties using native CSS animations, if possible. Uses the original jQuery.fn.animate() otherwise.

    API

    $.fn.animate(props, speed, callback) -> jQuery
    {Object}

    The CSS properties to animate

    {optional:Integer|String|Object} defaults to 400

    The animation duration in ms. Will use jQuery.fn.animate if a string or object is passed

    {optional:Function}

    A callback to execute once the animation is complete

    {jQuery}

    The jQuery element
