dropend jQuery.event.special.dropend

 

dropend is called when the drag motion is done for this drop element. The event handler gets an instance of jQuery.Drag passed as the second and a jQuery.Drop as the third parameter.

 $('.droparea').on('dropend', function(ev, drop, drag) {
     // Remove the drop element highlight
     $(this).removeClass('highlight');
 });
