findAllData can.Model.findAllData

 

Type Definition: {function(params)}

Parameters

  1. params {Object}

    Specifies the list to be retrieved.

Returns

{can.Deferred}

A deferred that resolves to a data structure that can be understood by can.Model.models.

Retrieves a list of items for can.Model.models, typically by making an Ajax request.

Use

Typically, findAll is implemented with a "string" or ajax settings object like:

findAll: "GET /tasks"

or

findAll: {url: "/tasks", dataType: "custom"}
