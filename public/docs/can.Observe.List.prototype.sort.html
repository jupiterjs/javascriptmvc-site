sort can/observe/sort

 

list.sort(sortfunc)

Sorts the instances in the list.

var list = new can.Observe.List([
    { name: 'Justin' },
    { name: 'Brian' },
    { name: 'Austin' },
    { name: 'Mihael' }])

list.comparator = 'name';
list.sort(); //- sorts the list by the name attribute
