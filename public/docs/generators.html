Generators generators

 

JavaScriptMVC comes with the following generators:

js jmvc/generate/app path

Parameters

  1. path {String}

    The lowercase directory your application will be placed within. Keep application names short because they are used as namespaces. The last part of the path will be taken to be your application's name.

    Creates a JavaScriptMVC application structure.

js jmvc/generate/control app/controls/video

Parameters

  1. app/controls/video {String}

    The moduleId name of your control.

    Creates a can.Control and test files.

js jmvc/generate/model app/models/name

Parameters

  1. `app/models/name` {String}

    The name of the model.

    Creates a can.Model and test files.

js jmvc/generate/page path/to/app path/to/page.html

Parameters

  1. path/to/app {String}

    The path to your apps folder.

  2. path/to/page.html {String}

    The path to the page you want to create.

    Creates a page that loads steal.js and an application.

js jmvc/generate/plugin path/to/plugin

Parameters

  1. path/to/plugin

    The path to where you want your plugin.

    Create a file and folder structure for a basic jQuery plugin.

js jmvc/generate/scaffold app/models/modelname

Parameters

  1. app/models/modelname {String}

    The moduleId of you want to create CRUD functionality for.

    Creates the controls, models, and fixtures used to provide basic CRUD functionality.
