steal

 

steal(moduleIdRef..., definition(module...) )

Loads scripts, css, and other modules into your application.

steal('jquery','can','./app.less',function($, can){

})

Parameters

  1. moduleIdRef {moduleIdReference}

    Specifies the module dependencies of the current module. A moduleIdReference uses:

    • The moduleId of the module depending on the moduleIdReference
    • steal.config's map and root configuration options

    too look up a moduleId.

    The moduleId uses steal.config's paths option to determine the file location or url of the module and the file extension and other steal.config values to determine how to process the module to retrieve a value.

  2. definition {definition(moduleValue)}Optional

    A callback function that gets passed the module values of the modules referenced by the moduleIdReferences and whose return value defines the value of the current module.

Returns

{steal(moduleIdRef, definition)}

returns steal for chaining.
