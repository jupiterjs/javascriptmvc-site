jQuery.Event.prototype.keyName jquery/event/key

 

Returns a string representation of the key pressed:

 $("input").on('keypress', function(ev){
     if(ev.keyName() == 'ctrl') {
         $(this).addClass('highlight');
     }
 });

The key names mapped by default can be found in the jQuery.event.key overview.
