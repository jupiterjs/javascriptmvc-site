ghost

 

drag.ghost( [parent] ) clones the element and uses it as the moving element, leaving the original dragged element in place. The parent option can be used to specify where the ghost element should be temporarily added into the DOM. This method should be called in "draginit".

$("#todos").on(".handle", "draginit", function( ev, drag ) {
  drag.ghost();
})
