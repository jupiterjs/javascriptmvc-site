jQuery.Event.prototype.pause

 

event.paused() pauses an event (to be resumed later):

 $('.tab').on('show', function(ev) {
     ev.pause();
     // Resume the event after 1 second
     setTimeout(function() {
         ev.resume();
     }, 1000);
 });
