requestHandler can.fixture.types.requestHandler

 

Specifies the response of a fixture. Used in can.fixture.

Type Definition: {function(request, response)}

Parameters

  1. request {can.AjaxSettings}

    The ajax settings object that was passed to can.ajax or a jQuery ajax method. Any templated portions of the url passed to can.fixture are added as data to request.data. For example, calling:

    GET /todos/5

    With the following fixture:

    can.fixture("/todos/:id",function(request, response){

})

    request.data.id will be `5.

  2. response {responseHandler()}Optional

    Optionally called to specify the response of the fixture.

Returns

{* | undefined}

If a value is returned, it is used as a JSON response body. If nothing is returned, it's expected that responseHandler was used.
