can/util/bind

 

Type Definition: {Object}

  • bind {function()}
  • unbind {function()}

Provides mixin-able bind and unbind methods. bind() calls this._bindsetup when the first bind happens and. unbind() calls this._bindteardown when there are no more event handlers.
