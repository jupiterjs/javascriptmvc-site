open

 

steal.build.open(url, [stealData, cb])

Parameters

  1. url {String}

    The html page to open.

  2. stealData {Object}Optional

    Data to configure steal with.

  3. cb {function(openerObject)}

    An object with properties that makes extracting the content for a certain tag slightly easier.

  4. includeFns=true {Boolean}Optional

    indicates that iteration should happen

Returns

{Object}

an object with properties that makes extracting the content for a certain tag slightly easier.

steal.build.open(url, [stealData], cb(opener) )
opens a page that typically uses steal.js. Once all scripts have been loaded, cb is called back with opener. opener is an object that helps walk through the modules loaded by steal.

steal.build.open(
  "page.html",
  {
       startId: "myapp"
  }, 
  function(opener){})

open opens a page by:

  • temporarily deleting the rhino steal
  • opening the page with Envjs
  • setting back rhino steal, saving envjs's steal as steal._steal;
