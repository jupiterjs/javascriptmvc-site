jQuery.Drop jquerypp/event/drop

 

The jQuery.Drop constructor is never called directly but an instance is passed to the to the dropinit, dropover, dropmove, dropon, and dropend event handlers as the third argument (the second will be the jQuery.Drag):

 $('#dropper').on('dropover', function(el, drop, drag) {
     // drop -> $.Drop
     // drag -> $.Drag
 });
