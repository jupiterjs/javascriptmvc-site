paths

 

steal.config( "paths", pathConfig )

Maps moduleIds to paths. This is used to override steal.idToUri. Often, this can be used to specify loading from a CDN like:

steal.config({
  paths: {
    "jquery" : "http://cdn.google.com/jquery"
  }
});

To keep loading jQuery in production from the CDN, use steal.config.shim and set the "exclude" option.
