leave

 

hover.leave(delay) sets a delay for how long the hover should stay active after the mouse left. This method should only be used in [jQuery.event.hover.hoverinit hoverinit]:

 $('.hoverable').on('hoverinit', function(ev, hover) {
     // Stay active for another second after the mouse left
     hover.leave(1000);
 });
