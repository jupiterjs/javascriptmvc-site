moveToPoint

 

range.moveToPoint(point) moves the range end and start position to a specific point. A point can be specified like:

 //client coordinates
 {clientX: 200, clientY: 300}

 //page coordinates
 {pageX: 200, pageY: 300}
 {top: 200, left: 300}
