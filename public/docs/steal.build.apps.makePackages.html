makePackages

 

makePackages(options, buildOptions)

Parameters

  1. options {Object}

  2. buildOptions {Object}

Creates packages that can be downloaded.

Recursively uses getMostShared to pull out the largest sharing. It makes a package of the sharing and marks the apps that need that sharing.

The apps that need the sharing packages are mostly dummy things. A production file might steal multiple packages.

say package A and package B

say package A has jQuery

so, the production file has code like: steal('jquery') It needs to know to not load jQuery this is where has comes into place

steal({id: 'packageA', has: 'jquery'})

This wires up steal to wait until package A is finished for jQuery. So, we need to know all the packages and app needs, and all the things in that package.
