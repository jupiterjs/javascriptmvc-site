can.isFunction

 

Check if an Object is a function.

can.isFunction(obj)

Parameters

  1. obj {Object}

    The object to check.

Returns

{Boolean}

Whether obj is a function.

can.isFunction(object) returns if an object is a function similar to jQuery.isFunction.

can.isFunction({})           //-> false
can.isFunction(function(){}) //-> true
