distance jQuery.Hover.distance

 

Property Type: {Number}

$.Hover.distance is the maximum distance (in pixels) that the mouse is allowed to travel within the time of jQuery.Hover.delay in order to activate a hover.

Set this value as a global default. The default is 10px.

 // Set the global hover distance to 1 pixel
 $.Hover.distance = 1;
