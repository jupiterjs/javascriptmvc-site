ext

 

steal.config("ext", extensionConfig)

Configures processing behavior of moduleId extensions. For example:

steal.config("ext",{
  js: "js",
  css: "css",
  less: "steal/less/less.js",
  mustache: "can/view/mustache/mustache.js"
})

extensionConfig maps a filename extension to be processed by a type (like js: "js") or to a dependency moduleId that defines that type (like less: "steal/less/less.js").
